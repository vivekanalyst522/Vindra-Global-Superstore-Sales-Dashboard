# Vindra-Global-Superstore-Sales-Dashboard

📊 Vindra Superstore Sales Dashboard

This project is an interactive Excel Sales Dashboard designed to analyze and visualize retail sales data. It provides valuable insights into sales performance across different months, product categories, customer age groups, cities, states, and sales channels.

🔹 Key Features:
• Monthly Sales Analysis
• Category-wise Sales Distribution
• Age Group Sales Comparison
• Top 10 Cities Sales Analysis
• Top 5 States Sales Performance
• Channel-wise Sales Trends
• Interactive Filters and Slicers

🛠 Tools Used:
• Microsoft Excel
• Pivot Tables
• Pivot Charts
• Slicers
• Data Visualization

📈 Dashboard Insights:
• Identify top-performing cities and states.
• Track monthly sales growth and trends.
• Compare sales across customer age groups.
• Analyze category and channel performance.
• Support better business decision-making through data visualization.

This dashboard demonstrates how Excel can be used to transform raw sales data into meaningful business insights.
